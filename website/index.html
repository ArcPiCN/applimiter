三步开始

不需要 root，普通安卓 8.0 及以上设备即可。

授权无障碍服务

系统设置 → 无障碍 → 戒刷止刷页面监控 → 开启。

授权悬浮窗（可选但推荐）

用于在任意应用快速添加规则。从应用首页一键跳转授权页面。

添加或导入规则

使用内置推荐规则、悬浮窗一键添加，或从 JSON 文件导入。

常见问题

如果有未列出的问题，欢迎到 GitHub 提 Issue。

它会上传我的数据吗？

不会。所有规则和运行数据都保存在本机 Room 数据库里，应用没有联网上报逻辑。源码完全开放，可自查。

为什么需要无障碍权限？

需要监听前台 Activity 名才能识别"现在是不是某个被限制的页面"。不会读取页面内容、不会模拟点击。

时间到了如何强制退出？

触发 GLOBAL_ACTION_HOME 直接返回桌面，比 BACK 更可靠，不会被应用拦截。

国产机后台被杀怎么办？

请到系统的"后台管理 / 自启动"里允许戒刷止刷常驻，无障碍服务才能一直工作。

能贡献规则吗？

当然。fork 仓库后修改 app/src/main/res/raw/default_rules.json 提 PR 即可。
